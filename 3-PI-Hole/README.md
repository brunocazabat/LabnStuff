# PI-HOLE

## Description

This is a simple project to install and configure a Pi-Hole server on a Proxmox LXC container.

Core: 1

Mem: 512MB

Swap: 0MB

Disk: 8GB

## Installation

Follow the commands from the `LXC-pihole.bash` file.
