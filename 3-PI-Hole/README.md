# PIHOLE

## Description

This is a simple project to install and configure a Pi-Hole server on a Proxmox LXC container.

Core: 1
Mem: 512MB
Swap: 512MB
Disk: 8GB

## Installation

Follow the cmd's from the `LXC-pihole.bash` file.
